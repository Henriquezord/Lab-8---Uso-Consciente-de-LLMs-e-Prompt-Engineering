## Descrição da Atividade
Este repositório contém os artefatos gerados para a atividade prática "Lab 8", que explora o uso consciente de Large Language Models (LLMs) e técnicas de *Prompt Engineering*. O objetivo principal deste trabalho foi interagir com uma ferramenta de IA generativa para resolver pequenas tarefas académicas, avaliando criticamente as respostas geradas, refinando comandos (prompts) para obter resultados controlados e otimizados, e refletindo sobre as boas práticas do uso destas tecnologias na aprendizagem de computação.

## Estrutura do Repositório
- "README.md": Este arquivo com a descrição sumária do projeto.
- "relatorio.pdf': Documento final contendo todas as análises, códigos em C, testes práticos e reflexões críticas solicitadas no laboratório.
- "prompts.txt": Arquivo contendo a transcrição exata de todos os comandos enviados à IA durante a realização das tarefas.

## AI Usage Statement
Este trabalho utilizou uma ferramenta de IA generativa para auxiliar na geração de explicações e exemplos de código.
Todos os prompts utilizados estão documentados em "prompts.txt".
As respostas foram revisadas, testadas e verificadas pelo autor antes de serem incluídas no relatório.
